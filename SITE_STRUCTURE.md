## Quick Edit Guide

### To update your information:
- **Personal details**: Edit `data/en/author.yaml`
- **Add/remove skills**: Edit `data/en/sections/skills.yaml`
- **Update work history**: Edit `data/en/sections/experiences.yaml`
- **Modify projects**: Edit `data/en/sections/projects.yaml`
- **Change education**: Edit `data/en/sections/education.yaml`

### To change section visibility:
Edit `hugo.yaml` and modify the `params` section.

### To reorder sections:
Change the `weight` value in each section's YAML file (lower numbers appear first).
