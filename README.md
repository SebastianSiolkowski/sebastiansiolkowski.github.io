## Local Development

### Prerequisites

- Hugo Extended v0.146.0 or higher
- Go 1.21 or higher
- Node.js v18.x or later
- npm 8.x or later

### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/sebastiansiolkowski/sebastiansiolkowski.github.io.git
   cd sebastiansiolkowski.github.io
   ```

2. Initialize Hugo modules:
   ```bash
   hugo mod tidy
   ```

3. Generate and install node dependencies:
   ```bash
   hugo mod npm pack
   npm install
   ```

4. Run the development server:
   ```bash
   hugo server -w
   ```

5. Open your browser and navigate to `http://localhost:1313`
